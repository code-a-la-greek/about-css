## Flexbox
[Εδώ](https://youtu.be/a72nYJdCzNI) αναλύουμε το `flexbox` και τις ιδιότητές του.

### Flexbox Properties:
**Parent-element**
- display:flex
- flex-direction: row(default)/column/row-reverse/column-reverse
- flex-wrap: nowrap/wrap/wrap-reverse
- justify-content: 
    - start
    - end
    - stretch
    - center
    - space-around
    - space-between
    - space-evenly
- align-items: 
    - start
    - end
    - stretch
    - center
- gap

**Child-element**
- flex
